In src/app/(dashboard)/layout.tsx:

1. Add a skip-to-content link as the FIRST child inside SidebarProvider, before the Sidebar element:
   ```
   <a href="#main-content" className="skip-to-content">
     Skip to content
   </a>
   ```

2. Add `id="main-content"` and `tabIndex={-1}` to the existing `<main>` element so it can receive focus:
   ```
   <main id="main-content" tabIndex={-1} className="flex-1 p-6 outline-none">
   ```
   The `outline-none` prevents a visible outline when the main element receives programmatic focus.

3. Add `aria-label="Main navigation"` to the Sidebar component (or wrap the SidebarMenu in a `<nav aria-label="Main navigation">` if Sidebar doesn't accept aria-label -- check the component). The Sidebar component renders a div, so add the aria-label prop to it. If the shadcn Sidebar does not forward aria-label, wrap the SidebarGroup's SidebarGroupContent in `<nav aria-label="Main navigation">`.

4. Add `role="banner"` to the header element (it already is a `<header>` so this is implicit, but ensure the header has `aria-label="Top bar"` for specificity since there may be multiple landmarks).

In src/app/globals.css:

5. Add a `.skip-to-content` class in the `@layer base` block:
   ```css
   .skip-to-content {
     @apply sr-only focus:not-sr-only focus:fixed focus:top-4 focus:left-4 focus:z-50 focus:rounded-md focus:bg-background focus:px-4 focus:py-2 focus:text-sm focus:font-medium focus:text-foreground focus:shadow-lg focus:ring-2 focus:ring-ring focus:outline-none;
   }
   ```
   This makes the skip link invisible until focused via keyboard Tab, then it appears as a floating button in the top-left corner.

6. Add a global focus-visible style to ensure ALL interactive elements have visible focus rings, including plain `<a>` tags and custom elements that may not have shadcn's built-in styles. In the `@layer base` block, add:
   ```css
   a:focus-visible {
     @apply rounded-sm outline-2 outline-offset-2 outline-ring;
   }
   ```
   This catches Link-wrapped cards and other anchor elements that lack explicit focus styles.
