**src/app/api/contacts/route.ts** -- List and Create (TDD green phase):

Follow the exact pattern from `src/app/api/settings/profile/route.ts`.

GET handler:
1. `createClient()`, `getUser()`, 401 if not auth
2. Read `searchParams`: `category` (optional filter), `search` (optional text search)
3. Build query: `supabase.from("contacts").select("*, action_items(id, text, completed)").eq("user_id", user.id).order("updated_at", { ascending: false })`
4. If `category` param: add `.eq("category", category)`
5. If `search` param: add `.or(\`name.ilike.%${search}%,email.ilike.%${search}%,company.ilike.%${search}%,notes.ilike.%${search}%\`)`
6. Execute and return `{ data }`

POST handler:
1. Auth check
2. Parse body with `contactSchema.parse(body)`
3. Insert into contacts with `user_id: user.id, ai_confidence: "manual"`
4. Return `{ data }` with status 201

**src/app/api/contacts/[id]/route.ts** -- Detail, Update, Delete:

GET handler:
1. Auth check
2. Fetch contact with full joins: `supabase.from("contacts").select("*, action_items(*), contact_meetings(meeting_id, meetings(id, title, meeting_date, summary, granola_url))").eq("id", params.id).eq("user_id", user.id).single()`
3. Return 404 if not found, `{ data }` if found

PUT handler:
1. Auth check
2. Parse body with `contactUpdateSchema.parse({ ...body, id: params.id })`
3. Update contact: `supabase.from("contacts").update({ ...parsed, updated_at: new Date().toISOString() }).eq("id", params.id).eq("user_id", user.id)`
4. Return 404 if no rows affected, `{ success: true }` if updated

DELETE handler:
1. Auth check
2. Delete: `supabase.from("contacts").delete().eq("id", params.id).eq("user_id", user.id)`
3. Return `{ success: true }`

**src/app/api/contacts/[id]/action-items/route.ts** -- Toggle action item completion:

PUT handler:
1. Auth check
2. Parse body: `{ actionItemId: string, completed: boolean }`
3. Update: `supabase.from("action_items").update({ completed, updated_at: new Date().toISOString() }).eq("id", actionItemId).eq("user_id", user.id)`
4. Return `{ success: true }`

All routes use `import { NextResponse } from "next/server"` and `import { createClient } from "@/lib/supabase/server"`. All use ZodError catch for 400 responses.
