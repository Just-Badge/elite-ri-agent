Build the AI draft generation engine and Trigger.dev scheduling pipeline: a layered-context AI prompt module that generates personalized outreach emails via z.ai GLM5, an hourly cron dispatcher that identifies users needing draft generation, and a per-user task that finds due contacts, generates drafts, persists to DB, and best-effort syncs to Gmail.

Purpose: This is the core engine that activates the "meeting happened -> relationship maintained" loop. Without it, contacts accumulate but no outreach happens.
Output: 3 production modules (AI drafter, dispatcher, per-user task), 3 test files

    3. Implement `src/trigger/generate-user-drafts.ts`:
       - import { task, logger } from "@trigger.dev/sdk"
       - import { createAdminClient } from "@/lib/supabase/admin"
       - import { decrypt } from "@/lib/crypto/encryption"
       - import { generateOutreachDraft } from "@/lib/ai/draft-outreach"
       - import { createGmailDraft } from "@/lib/gmail/client"
       - import { fetchOpenBrainContext } from "@/lib/open-brain/client"

       - export const generateUserDrafts = task({
           id: "generate-user-drafts",
           retry: { maxAttempts: 3, factor: 1.8, minTimeoutInMs: 500, maxTimeoutInMs: 30000 },
           run: async (payload: { userId: string }) => {
             1. Fetch user settings (personality_profile, business_objectives, projects, zai_api_key_encrypted)
             2. Decrypt zai API key
             3. Query contacts due for outreach:
                SELECT contacts WHERE user_id = payload.userId AND status = 'active' AND email IS NOT NULL
                AND outreach_frequency_days IS NOT NULL
                AND (last_interaction_at IS NULL OR last_interaction_at + interval (outreach_frequency_days || ' days') < NOW())
                AND NOT EXISTS (SELECT 1 FROM outreach_drafts WHERE contact_id = contacts.id AND status = 'pending_review')
                ORDER BY last_interaction_at ASC NULLS FIRST LIMIT 20
             4. For each due contact (sequentially with 200ms delay):
                a. Fetch recent meetings: contact_meetings JOIN meetings, last 5, ordered desc
                b. Fetch action items: WHERE contact_id = contact.id AND completed = false
                c. Fetch Open Brain context: fetchOpenBrainContext(userId, contact.name, contact.email)
                d. Build DraftContext with all 5 layers
                e. Call generateOutreachDraft(decryptedApiKey, context)
                f. INSERT into outreach_drafts: user_id, contact_id, subject, body, ai_rationale, status='pending_review', gmail_sync_status='pending'
                g. Best-effort Gmail sync: try { gmailDraftId = await createGmailDraft(userId, contact.email, subject, body); UPDATE outreach_drafts SET gmail_draft_id, gmail_sync_status='synced' } catch { UPDATE gmail_sync_status='failed'; logger.warn }
                h. 200ms delay before next contact
             5. Return { generated: count, gmailSynced: count, gmailFailed: count }
           }
         })
  </action>
